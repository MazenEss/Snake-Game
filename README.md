# Snake-Game
A classic snake game developed Using Python Turtle module.

<img src="./Screenshots/Snake Game.jpg">

## New Features
- Supported (ARROWS,WASD) for moving.
  
## Technologies Used
- Turtle Module in Python.
  
## Setup
- Download EXE file and Start Playing.
